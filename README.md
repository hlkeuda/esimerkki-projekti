# Esimerkki-projekti

Tämä on yksinkertainen Java-projekti Git-harjoituksia varten.

## Sisältö

- `Tervehdys.java` - Yksinkertainen tervehdysohjelma

## Käyttö

```bash
# Käännä
javac Tervehdys.java

# Suorita
java Tervehdys
```

## Harjoituksen tarkoitus

Tämä projekti on tarkoitettu harjoitteluun:
- Kloonaa tämä projekti
- Tutki koodia
- Kokeile git pull -komentoa
- Kokeile git push -komentoa (ei toimi ilman oikeuksia)
- Fork:aa projekti ja kokeile uudelleen

## Opettajalle

Voit päivittää tätä projektia harjoituksen aikana, jotta opiskelijat voivat harjoitella `git pull` -komentoa.
